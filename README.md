# 関東令和の乱シミュレーター

千葉・東京・埼玉・茨城・神奈川の自治体が、兵力を振り分けながら関東の覇権を争う、観戦型の戦略シミュレーションです。

## 概要

- 関東周辺の自治体をタイルとして配置
- 全勢力がCPUとして自動で行動
- 1ターンは1カ月、2026年4月から進行
- 攻撃・移動・占領・降伏の結果がリアルタイムで地図に反映
- 地面の色で現在の勢力範囲を表示
- マウスホイールでズーム、ドラッグでスクロール
- BGM・効果音つき

## 遊び方

`index.html` をブラウザで開くと開始します。

全プレイヤーはCPUなので、基本的には戦況を観戦するゲームです。  
画面上部には年月、勢力ごとの自治体数、現在のバトル結果が表示されます。

## 主なルール

- 各自治体は兵力を持つ
- 攻撃側と防衛側が兵力ぶんのサイコロを振り、合計値が高い方が勝利
- 攻撃成功時、攻撃元には兵力1を残し、残りが占領先へ移動
- 占領後、兵力が残っていればさらに隣接自治体へ連続侵攻できる
- 県庁所在地を占領された勢力は降伏
  - 千葉: 千葉市
  - 東京: 新宿区
  - 埼玉: さいたま市
  - 茨城: 水戸市
  - 神奈川: 横浜市
- 攻撃フェーズ後、攻撃を見送った自治体のみ兵力移動できる
- 各勢力は攻撃・移動をそれぞれ最大20回まで実行
- 兵力の自然回復上限は人口規模に応じる
- 移動による一時的な兵力上限は20

## CPUの思考

CPUは以下を優先します。

- 敵の首都を攻撃する
- 敵首都に近い自治体を攻める
- 自分の首都周辺の守りを固める
- 前線へ兵力を集める

## ファイル構成

- `index.html`: 画面構造
- `styles.css`: 見た目
- `app.js`: ゲーム本体
- `bgm/`: 効果音
- `geojson-cache/`: 地形・自治体配置用のGeoJSONキャッシュ
